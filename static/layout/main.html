About this site

If you landed here looking for someone named Brandon Lind and he isn’t a software architect / developer, you’re probably in the wrong place ;).

I use this site as a sandbox to keep current with emerging technologies and theories in web architecture and software development. As such, it’s usually pretty plain-vanilla, but under the covers, I’m practicing everything from developing using a cloud IDE (such as Cloud9or Koding) to using Github and deploying to Heroku.

The server software also changes constantly. Sometime this site is developed using Ruby on Rails, sometimes Nodejs. Typically for a persistance engine, I stick with MongoDB.

The JavaScript framework also changes constantly, from Backbone to Knockout, Angular to KendoUI, it’s always in flux and a majority of my changes are to this area.

Terminal Aviation

Terminal Aviation is a fictional company which owns and operates an online collaboration portal and marketplace. The goal is to connect aircraft owners with people looking to charter a flight.

Future expansion modules:

  • Aircraft mechanics & maintenance companies will be able to bid on requests for repairs & maintenance contracts requested by aircraft owners or aircraft management companies.
  • Aircraft management companies will be able to bid on contacts requested by aircraft owners.
  • Charter companies will be able to bid on charter management contracts requested by aircraft owners.
